GitHub Profile Finder

A simple web application that allows users to search and view GitHub profiles using the GitHub REST API.
Users can enter a GitHub username to see profile details and public repositories.

🚀 Features

🔍 Search GitHub users by username

👤 Display user profile information

📦 View public repositories

⭐ Show repository stars

🧾 Display repository descriptions

❌ Error handling for invalid usernames

🖼️ Preview

Example search:

Username: torvalds

Shows profile of Linus Torvalds (creator of Linux).
